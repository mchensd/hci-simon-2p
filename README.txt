First, install pyenv and pipenv if you don't already have them.

Install Pyenv: https://github.com/pyenv/pyenv#basic-github-checkout

Install pipenv: https://pypi.org/project/pipenv

Run pipenv install to install required packages.

To start the system, run pipenv run python3 src/main.py --websocket-server 172.28.142.145:8888 --mode record
